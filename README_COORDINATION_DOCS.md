## 🔑 Key Concepts

### WorkSession
Long-running session where specialists iterate 50+ times with accumulated context.

### Iteration
Structured work unit with input, processing, output, and reflection.

### Finding
Structured result (not just text) with type, evidence, and metadata.

### Checkpoint
Snapshot for pause/resume enabling human-in-the-loop workflows.

### Feedback Loop
Real-time communication from orchestrator to agent during iteration.

## 🎯 Success Criteria

- ✅ Agents can iterate 50+ times
- ✅ Context accumulates between iterations
- ✅ Real-time feedback works
